## 프로젝트 소개

ITerview는 포스트 코로나로 변화된 채용시장에 적합한 면접 준비를 위한 서비스입니다. 기존에 존재하던 모의 면접 서비스들과는 다르게 직군별로 세세한 면접 질문을 접할 수 있고, 비대면 면접과 유사한 환경에서 모의 면접을 경험할 수 있습니다. 모의 면접 서비스에 AI 피드백과 모범 답안 기능을 추가하여 사용자가 타자의 도움 없이 혼자 면접을 준비할 수 있도록 서비스를 개발하였습니다. ITerview를 사용함으로써 수많은 취업 준비생 사이에서 자신의 경쟁력을 더 키울 수 있습니다.

## 핵심 기능 📦

### ⭐️ 선택형 or 랜덤형으로 원하는 직무의 질문으로 구성
<img width="932" alt="image" src="https://github.com/BLACKPINK-SLJY/.github/assets/94633589/c2d14ffc-f74e-41ad-b9ca-88423da25c80">

- 여러 가지 면접 질문 중에 자신이 원하는 질문을 선택하여 모의 면접을 볼 수 있어요.
- 해당 직무의 면접 질문들을 랜덤으로 테스트해볼 수 있어요.
- 면접 질문 리스트를 카테고리 별(최신, 난이도 낮은, 난이도 높은, 인기)로 정렬해서 볼 수 있어요.

### ⭐️ 질문 검색 기능
<img width="932" alt="image" src="https://github.com/BLACKPINK-SLJY/.github/assets/94633589/e3b902dc-9019-41d5-bc39-ad17ba536573">

- 사용자가 원하는 질문을 빠르게 찾을 수 있는 질문 검색할 수 있어요.

### ⭐️ 타이머와 함께 면접 녹화하기
![image](https://github.com/BLACKPINK-SLJY/.github/assets/94633589/34241122-8105-4cc5-a035-8a7ed4790539)

- 면접을 시작하면 5초 카운트 다운 후, 바로 녹화가 시작돼요.
- 녹화가 시작되고 난 후, 1분 단위로 얼마나 시간이 지났는지 알려줘요.

### ⭐️ 정답 키워드 제공
<img width="932" alt="image" src="https://github.com/BLACKPINK-SLJY/.github/assets/94633589/a3d7d37a-8807-45d7-b6dd-1cd81502541d">

- 면접 영상 제출 후에 사용자가 미흡했던 점을 빠르게 짚을 수 있도록 모범 답안 키워드를 제공해요.

### ⭐️ 내가 푼 문제 or 북마크 문제 확인
<img width="932" alt="image" src="https://github.com/BLACKPINK-SLJY/.github/assets/94633589/b87dee6d-7715-4f81-a022-cc374bd5d1d7">

- 마이 페이지에서 사용자가 푼 문제와 북마크해 둔 문제들을 확인할 수 있어요.
- 면접 질문 리스트를 카테고리 별(최신, 난이도 낮은, 난이도 높은, 인기)로 정렬해서 볼 수 있어요.

### ⭐️ 내 답변 분석
<img width="932" alt="image" src="https://github.com/BLACKPINK-SLJY/.github/assets/94633589/aa37ad03-dfb0-44e3-bc68-7b01ccdda588">

- Amazon Transcribe 서비스를 이용하여 STT(Speech To Text)로 변환한 내 답변을 확인해볼 수 있어요.
- 부정확하게 답변된 부분에는 밑줄로 표시가 돼서 사용자의 명확한 발음이 필요한 부분을 확인할 수 있어요.
- 추출된 텍스트를 기반으로 AI가 사용자의 답변을 평가해주고 평가 점수와 함께 피드백 및 모범 답안을 제시해줘요.

### ⭐️ 질문 답변 스크립트
![image](https://github.com/BLACKPINK-SLJY/.github/assets/94633589/b6422beb-93fe-48a4-b70e-7a7000df2ff4)

- 녹화된 답변 영상과 함께 타임스탬프로 간편하게 VTT 형식의 답변 스크립트를 확인해볼 수 있어요.
